a bit more about this project

this project was meant to be my attempt at making students entertained by things that are school related in this hellish time at the end of the semester.

we're all tired, sleep deprived, and that makes things get WEIRD - i aimed to capture that type of exhausted disorientation in a humorous, uncanny way so we can treat it like just another interactive game.

I hope that in this short experience, at least some things made you laugh ^-^'
